# البدء مع منشئ الصفحات

## المتطلبات

- حساب ترميز OS نشط
- صلاحية الوصول إلى الصفحات (تأكد مع مدير حسابك)
- ميزة منشئ الصفحات مفعّلة لمنشأتك

## الوصول إلى منشئ الصفحات

1. سجّل الدخول إلى لوحة تحكم ترميز OS
2. من القائمة الجانبية، ابحث عن قسم "الصفحات"
3. انقر على "الصفحات" للوصول إلى قائمة صفحات منشأتك

## واجهة الصفحات — نظرة عامة

ستجد في صفحة "الصفحات":
- **قائمة الصفحات**: تعرض جميع صفحات منشأتك مع حالتها (مسودة / منشورة / مؤرشفة)
- **زر "صفحة جديدة"**: لإنشاء صفحة جديدة
- **فلاتر**: تصفية الصفحات حسب الحالة
- **بحث**: البحث عن صفحة بالاسم أو الرابط

## إنشاء أول صفحة

1. انقر على زر **"صفحة جديدة"** في أعلى الصفحة
2. اختر بين:
   - **من قالب**: ابدأ بقالب جاهز بمحتوى عربي أصيل
   - **صفحة فارغة**: ابدأ من الصفر
3. أدخل اسم الصفحة والرابط
4. انقر "إنشاء وفتح المحرر"

## واجهة المحرر

بعد فتح المحرر ستجد:
- **منطقة الصفحة** (المركز): حيث تُبنى الصفحة
- **لوحة البلوكات** (اليمين): قائمة البلوكات التي يمكن إضافتها
- **لوحة الخصائص** (اليسار): تعديل خصائص البلوك المحدد
- **شريط الأدوات** (الأعلى): نشر، حفظ، SEO، معاينة

## الخطوة التالية

اقرأ دليل إنشاء الصفحات لمعرفة كيفية التعامل مع المحرر بالتفصيل.
