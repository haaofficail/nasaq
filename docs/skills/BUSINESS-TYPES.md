# أنواع الأعمال — كل نوع له متطلبات مختلفة

## القاعدة الذهبية:
- النظام يكتشف نوع البيزنس ويعرض الحقول المناسبة فقط
- لا حقول زيادة — لا حقول ناقصة
- المخزون نوعين: مستهلكات (تنقص) + أصول (ما تنقص)

## صالون:
- الخدمات: اسم + مدة + سعر + المختصة
- المخزون مستهلكات: صبغة، شامبو، أوكسجين — تنقص تلقائي مع كل خدمة (وصفة/recipe)
- الحجز: اختر وقت → اختر مختصة → اختر خدمة → تأكيد

## مقهى/مطعم:
- المنتجات: اسم + سعر + وصفة (مكونات + كميات)
- المخزون مستهلكات: قهوة، حليب، سكر — تنقص حسب الوصفة
- المخزون أصول: ماكينة قهوة، طاحونة — ما تنقص، بس تحتاج صيانة دورية
- الطلب: اختر من القائمة → ادفع (خطوتين بس)

## تأجير (خيام/كوش/طاولات — محفل):
- الأصول: اسم + حالة (ممتاز/جيد/يحتاج صيانة) + صور + موقع تخزين
- كل أصل بعد الاستخدام يحتاج: تنظيف + فحص حالة + صور قبل/بعد
- الكوشة ممكن تحتاج رش بويه حسب طلب العميل = خدمة إضافية مرتبطة بالأصل
- استلام وتسليم: checklist + صور + حالة + توقيع العميل
- بعض الأصول تحتاج صيانة دورية (كل 10 استخدامات مثلاً)

## محل ورد:
- المنتجات: ورد موسمي + باقات جاهزة + تنسيقات مخصصة
- المخزون مستهلكات: ورد (ينقص مع كل باقة) — فيه تاريخ انتهاء (ورد يذبل)
- المخزون أصول: أوعية، سلال — ما تنقص
- Hybrid: ممكن يبيع ورد + يأجر كوش وطاولات استقبال = بيع + تأجير في نفس النظام
- التوصيل: عنوان + وقت محدد + رسوم حسب المسافة

## فندق/شاليه:
- الوحدات: غرف + أجنحة + شاليهات — كل وحدة فريدة
- الحجز: تاريخ دخول + خروج + عدد ضيوف + خدمات إضافية
- المخزون مستهلكات: مناشف، صابون، شامبو — تنقص مع كل checkout
- تنظيف إلزامي بعد كل checkout — يمنع الحجز حتى يتم التنظيف

## مساج/سبا:
- الخدمات: نوع المساج + المعالج + المدة + منطقة الجسم
- المخزون مستهلكات: زيوت (لافندر، أرغان، جوز هند) — تنقص حسب نوع الجلسة
- الحجز: اختر خدمة → اختر معالج → اختر وقت (3 خطوات)

## جوالات/إلكترونيات:
- المنتجات: اسم + IMEI/Serial + ضمان + حالة (جديد/مستعمل/مجدد)
- كل قطعة فريدة — serial tracking مو كميات
- البيع: اختر منتج → ادفع → فاتورة + شهادة ضمان تلقائية

## القواعد المشتركة لكل الأنواع:
- عملاء + فواتير + تقارير + إعدادات = موجودة دائماً
- الفرق في: المنتجات/الخدمات + نوع المخزون + طريقة البيع/الحجز
- Business Type Registry يتحكم في كل شيء: المسميات + القوائم + الحقول
